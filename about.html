About This Website

The purpose of this website is to make it easier for people to find the recipe of sweet dish. People who make a lot of sweet dish at home often have to search a lot in order to learn how to make a sweet dish.

To make things easier I created this website so that people can easily find any type of indian sweet dish they want to make which makes things a lot easier. So they can utilise the remaining time making delicious sweet dishes!!!!!
